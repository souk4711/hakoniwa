# --seccomp

Set seccomp security profile [default: podman]

## audit

```console
$ hakoniwa run -vv --seccomp=audit
...
[..] UID mapping: container_id: [..]
[..] GID mapping: container_id: [..]
[..] Seccomp: Load 1 rules for architectures([..])
...
[..] Execve: "/bin/sh", []
...
```
